# Download

Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}

Add the dependencies

	dependencies {
	        compile 'com.colorssoftware.library:library:1.0.0'
	}
